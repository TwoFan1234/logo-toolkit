# 素材工具箱团队使用说明

这份说明是给只需要使用工具的同事看的，不需要安装 Python，也不需要打开 GitHub。

## 第一次使用

1. 从团队云盘下载最新的 `素材工具箱-v版本号-日期.zip`。
2. 右键压缩包，选择“全部解压”。
3. 打开解压后的文件夹，双击 `素材工具箱.exe`。
4. 如果 Windows 弹出安全提示，选择“更多信息”，再选择“仍要运行”。
5. 这是单文件版本，如果需要放到桌面，可以直接复制 `素材工具箱.exe`。

## 常用功能

- `批量处理图片`：批量改图片尺寸、格式等。
- `批量加 Logo`：给图片批量叠加 Logo。
- `批量视频处理`：压缩、转格式、裁剪、改尺寸、提取音频等。
- `批量加 EC`：给一整批视频统一叠加透明 MOV 结尾视频，只需要选择 EC 文件和重叠时长。
- `批量视频加 Logo`：给视频批量叠加 Logo。
- `视频套边框`：把视频叠到边框图片上，生成 16:9、1:1、9:16 等版本。

## V0.2 新增

### 批量加 EC

适合给一整批素材统一加结尾视频：

1. 先点左侧的 `批量加 EC`。
2. 导入要处理的视频素材。
3. 选择一个带透明通道的 EC 视频。
4. 设置 `重叠时长`。
5. 选择导出目录后开始处理。

建议先拿 1 到 3 条素材试跑，确认 EC 接入位置和导出效果没问题后，再跑整批。

## 推荐操作习惯

1. 先用 1 到 3 个测试文件跑一遍。
2. 确认预览和导出效果没问题后，再处理整批文件。
3. 导出目录建议单独新建，不要直接覆盖原文件。
4. 如果要覆盖原文件，请先备份。
5. 批量处理失败时，先看列表里的“说明”列。

## 更新版本

1. 删除或保留旧版本都可以，工具本身不需要安装。
2. 下载新的 zip。
3. 解压后双击新文件夹里的 `素材工具箱.exe`。
4. 不建议把新旧版本混在同一个文件夹里。

## 遇到问题时发给维护者的信息

请尽量发这几项：

- 你使用的工具版本，比如 `素材工具箱-v0.2-20260417`。
- 你点的是哪个功能。
- 出问题的文件格式，比如 `mp4`、`png`、`jpg`。
- 工具列表里显示的错误说明。
- 一张截图。
